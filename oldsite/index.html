What It Is

The WhirlyGlobe-Maply Component is a library you can use to put an interactive 3D globe or slippy map in your app. You can show imagery, networked tile data sets, labels, vectors, shapes, annotations and all sorts of other things.

WG-Maply is available for iOS as version 2.3. Android is currently in testing.

For a general overview of the features, watch the WhirlyGlobe and Maply videos. For more specific functionality, compile and run the WhirlyGlobeComponentTester app and run through the options.

WhirlyGlobe Video

WhirlyGlobe refers to the 3D interactive globe part of the toolkit. It shares most of its code with the Maply part.
